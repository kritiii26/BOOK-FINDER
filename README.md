📚 Book Finder Web Application-

Description-

This project is a web application that allows users to search for books and view details such as title, author, and cover image.

🎯 Purpose-

The purpose of this project is to demonstrate understanding of JavaScript, API integration using fetch, and UI development using CSS.

🔗 API Used

Google Books API 
https://developers.google.com/books/docs/v1/using




✨ Features

* Search books by name
* Display book details (title, author, cover image)
* Filter books by category
* Sort books by title or published date

🛠 Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API

🚀 How to Run

1. Clone the repository
2. Open index.html in your browser
